# Code

The `Code` component adds syntax highligting to your code snippets. You can use it as a block or inline.
Under the hood, it uses [highlight.js](https://highlightjs.org/).

## React

By default, `Code` doesn't include any language to reduce the size of the bundle. To install a new language,
you need to follow these steps:

1. Add highlight.js to your project

  <Code>
    {`yarn add highlight.js`}
  </Code>

1. Add the CSS of a dark theme to your `head`. We recommend the hybrid theme:

  <Code>
    {`<link rel="stylesheet" media="screen"
        href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/styles/hybrid.min.css">`}
  </Code>

1. Register the languages you need before use `Code`:

  <RegisterLanguage />

### Example

<Playground>
  <Code language="js">
    const num = 1;
  </Code>
  This is some <Code inline highlight={false}>console.log('inline JS code')</Code>
</Playground>

Since this is a generic component, it doesn't have any particular accessibility requirement.
It's just for presentation.
